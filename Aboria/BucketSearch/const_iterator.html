Description

A const iterator to a set of neighbouring points. This iterator implements a STL forward iterator type

const_iterator public construct/copy/destruct

  1. const_iterator();
    This constructor is used to return an end iterator. 
  2. explicit const_iterator(const BucketSearch * bucket_sort, const Vect3d centre, 
                        const int my_index = -1, const bool self = false);

    this constructor is used to start the iterator at the head of a bucket list

    Parameters:

    bucket_sort

    a pointer to the parent BucketSearch class

    centre

    the neighbourhood query point

    my_index

    (optional) the index of the query point

    self

    (optional) if true then assume that the user is finding all point pairs within a single container and so don't return each point pair twice (i.e. only 1/2 the normal neighbour checking required)
